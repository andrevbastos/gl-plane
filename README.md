# IFCG

## 📦 Dependencies

* CMake 2.8.5+
* OpenGL 3.3+
* GLFW3
* GLM
* IFCG

```bash
sudo apt-get install build-essential gpp cmake
sudo apt-get install libglfw3-dev libglm-dev
```

## 🔧 Compiling

Clone the repository:
```bash
git clone https://gitlab.com/oederaugusto/ifcg
cd ifcg
mkdir build
cd build
```

Compile and run the project:
```bash
cmake ..
make -j4
./bin/main-test
```
